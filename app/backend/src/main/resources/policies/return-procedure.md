# Procedura zwrotu sprzętu elektronicznego (przykładowa)

> Dokument przykładowy na potrzeby MVP. Treść jest fikcyjna i służy jako materiał wejściowy dla agenta przy ocenie **zwrotu**. Należy zastąpić rzeczywistą procedurą firmy.

## 1. Zakres
Procedura dotyczy zwrotu sprzętu elektronicznego (odstąpienie od umowy / zwrot towaru pełnowartościowego) i oceny, czy produkt można ponownie wprowadzić do sprzedaży.

## 2. Termin zwrotu
- Zwrotu można dokonać w ciągu **14 dni** od daty zakupu.
- Zgłoszenia po upływie 14 dni od daty zakupu są **odrzucane**.

## 3. Warunki przyjęcia zwrotu (decyzja: ZATWIERDŹ)
Zwrot przyjmuje się, gdy spełnione są łącznie:
1. Zgłoszenie nastąpiło w terminie (pkt 2).
2. Produkt nie nosi śladów użytkowania: brak rys, zabrudzeń, przetarć, śladów montażu/instalacji.
3. Produkt jest kompletny i w stanie pozwalającym na ponowną sprzedaż (jako nowy lub „open-box").

## 4. Przesłanki odrzucenia (decyzja: ODRZUĆ)
Zwrot odrzuca się, gdy występuje co najmniej jedna z przesłanek:
1. Widoczne ślady użytkowania (rysy, zabrudzenia, przetarcia, ślady palców/eksploatacji).
2. Uszkodzenie mechaniczne lub kosmetyczne obniżające wartość handlową.
3. Produkt niekompletny lub noszący ślady trwałej instalacji/personalizacji.
4. Upływ terminu zwrotu (pkt 2).

## 5. Przekazanie do weryfikacji człowieka (decyzja: ESKALUJ)
Sprawę kieruje się do przeglądu przez pracownika, gdy:
1. Zdjęcie jest nieczytelne lub nie przedstawia jednoznacznie stanu produktu.
2. Stan produktu jest graniczny (drobne, niejednoznaczne ślady — trudno ocenić zdatność do odsprzedaży).
3. Dane ze zgłoszenia są sprzeczne z obrazem.
4. Przypadek nie mieści się w niniejszej procedurze.

## 6. Następne kroki
- **Zatwierdzono:** przyjąć zwrot; oznaczyć produkt jako zdatny do ponownej sprzedaży (nowy / open-box); uruchomić zwrot środków zgodnie z procedurą.
- **Odrzucono:** przekazać klientowi uzasadnienie ze wskazaniem konkretnej przesłanki (np. ślady użytkowania) i poinformować o braku możliwości przyjęcia zwrotu.
- **Eskalacja:** zebrać lepsze zdjęcie / dodatkowe informacje i przekazać sprawę do osoby decyzyjnej.
